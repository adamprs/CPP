# C++ Modules

## Description

The C++ Modules project is a series of exercises designed to introduce and explore object-oriented programming in C++.

Through multiple modules, the project covers fundamental and advanced C++ concepts such as classes, encapsulation, inheritance, polymorphism, operator overloading, and templates. Each module focuses on specific features of the language, progressively building a deeper understanding of C++ and its paradigms.

The goal is to learn how to write clean, modular, and reusable code while mastering memory management, type safety, and the standard library.

This project emphasizes good coding practices, abstraction, and the differences between C and C++ programming.
